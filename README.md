# IP102 昆虫识别项目 / IP102 Insect Pest Recognition

- [中文使用说明](README_ZH.md)
- [English documentation](README_EN.md)

本项目实现 IP102 昆虫分类模型训练与对比，并支持使用已有 YOLO11 `best.pt` 对视频进行多目标检测。

This project implements IP102 classifier training and comparison, plus multi-object video detection using an existing YOLO11 `best.pt`.
